# read_write_go
This is a simple file read-and-write program written in Go. The program first writes text into a file and then reads the same file to display its contents. It demonstrates basic file handling in Go using os.WriteFile and os.ReadFile, along with simple error checking.

Go File Read & Write Demo
A beginner-friendly Go program that demonstrates how to:
- Write text to a file
- Read text from a file
- Handle errors properly
- Work with os.ReadFile and os.WriteFile

This was one of my first steps in learning Go file handling.
